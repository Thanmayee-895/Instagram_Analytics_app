Key Features
🔹 1. Dynamic Analytics Dashboard

Generates real-time influencer metrics based on username input using backend API logic.

🔹 2. FastAPI Backend

Built with FastAPI to handle REST API requests and return structured JSON analytics data.

🔹 3. Deterministic Data Generation

Uses hashlib and seeded random logic to ensure consistent results for the same username.

🔹 4. Sentiment Analysis Simulation

Displays Positive, Neutral, and Negative sentiment distribution in interactive pie charts.

🔹 5. Engagement Rate Calculation

Calculates and visualizes engagement metrics such as likes, comments, and interaction trends.

🔹 6. Interactive Data Visualizations

Uses Chart.js to render:

Pie Charts (Sentiment Distribution)

Line Charts (Engagement Trends)

Bar Charts (Interaction Types)

🔹 7. Hashtag Intelligence

Generates relevant hashtag suggestions dynamically based on user profile category logic.

🔹 8. Responsive UI Design

Fully responsive layout built using modern CSS styling for desktop and mobile compatibility.

🔹 9. REST API Architecture

Frontend communicates with backend using HTTP GET requests and JSON responses.

🔹 10. Scalable Design

Designed in a way that can be extended to integrate real Instagram Graph API in the future.
